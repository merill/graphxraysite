How it works

Graph X-Ray lets you view the Graph API and PowerShell log of actions taken in the Azure Active Directory and Intune portal.

Support & Feedback

This is an independently developed application and is not endorsed or supported by Microsoft.

A quick note. Chrome has a known issue where the Graph X-Ray panel doesn't show up in the Developer Tools panels list. To fix this you might need to close Dev Tools, refresh the page and then open Dev Tools (or repeat a few times).
